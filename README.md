# 🛍️ Trendify – Online Fashion Store & E-Commerce Web Platform

**Trendify** is a modern, full-stack web-based fashion e-commerce platform designed to provide a seamless online shopping experience with role-based access for customers, admins, and managers.  

This project was developed as part of the academic module **IT1100 – Internet and Web Technologies (Semester 2, Year 1)** for the BSc (Hons) in Information Technology program.

---

## 🌐 Overview

Trendify simulates a real-world e-commerce system where users can browse fashion products, manage shopping carts, and complete purchases securely. The system also includes powerful administrative and management features for handling products, users, orders, and system data.

The platform supports multiple fashion categories including:

- 👕 Men's Fashion  
- 👗 Women's Fashion  
- 🧒 Kids' Fashion  
- ✨ New Arrivals  

---

## 🚀 Core Features

### 👤 Guest Users
- Browse fashion products and categories  
- View product details  
- Access public pages (Home, About, Contact)  
- Register for an account  

---

### 🛒 Registered Customers
- Secure login and profile management  
- Search and filter products  
- Add/remove items from cart  
- Checkout and place orders  
- View order history  
- Manage account details  

---

### 🔐 Admin Panel
- Manage users and customer accounts  
- Add, update, and delete products  
- Control inventory and stock  
- Manage orders and payments  
- Oversee website content and system data  

---

### 📊 Manager Role
- View sales and performance reports  
- Monitor customer and order activity  
- Analyze business data and revenue trends  
- Support administrative decision-making  

---

## 🛠️ Technology Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (phpMyAdmin)  
- **Tools:** VS Code, XAMPP  
- **Architecture:** Modular MVC-style web application  
- **Concepts Used:** CRUD operations, session management, role-based access control
